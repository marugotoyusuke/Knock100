C LANGUAGE PRACTICE / 01-16

基礎プロ 100本ノック

01 から 16 までの学習範囲をもとにした、C言語の基礎練習問題集です。超基礎から少しずつ難しくなる順番に並べています。
特に指定がない限り、入力値が正しい形式で入力されるものとしてよいです。まずは「動くプログラムを書く」ことを優先してください。

HOW TO USE

使い方

  1. 各問題は knock00.c から knock99.c のようなファイル名で作成する。
  2. コンパイル例: gcc knock00.c -o knock00
  3. 実行例: ./knock00
  4. 表示文は完全一致でなくてもよいが、結果が分かるように表示する。
